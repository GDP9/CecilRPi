Cecil

CECIL assembly language grammar definition This grammar constitutes all the lexer rules employed by CECIL and corresponding parsing rules. It is a version of the CECIL Language originally developed by David Argles. it contains 39 simple instructions and user input datafields and labelfields. It catches the relevant errors thrown by the lexer and parser. The language used is Java.
